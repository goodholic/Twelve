# 🏰 GuildMaster - 길드 키우기

**장르**: 길드 시뮬레이션 + 턴제 자동 방치형 JRPG  
**플랫폼**: 모바일 (iOS/Android), PC(스팀)  
**화면**: 가로 모드 전용 (1920x1080 기준)  
**플레이**: 완전한 싱글플레이어 게임

## 🎮 게임 소개

모험가 길드를 설립하고 대륙 최고의 길드로 성장시키는 방치형 시뮬레이션 게임입니다.

### 핵심 특징
- **2:2 부대 전투**: 각 부대당 18명씩, 총 36명의 캐릭터로 정예 전투
- **길드 건설**: 3D 아이소메트릭 뷰로 길드 시설 건설 및 관리
- **캐릭터 수집**: 36명의 다양한 캐릭터를 수집하고 육성
- **완전 자동화**: 설정 후 자동으로 진행되는 방치형 시스템
- **싱글플레이**: 타인과의 경쟁 없이 자신만의 속도로 즐기는 게임

## 🎯 핵심 게임플레이

### 1. 소규모 정예 전투 시스템

```
플레이어 길드 (2부대)          vs          AI 적 길드 (2부대)
┌─────────────────┐                    ┌─────────────────┐
│  부대1 (6x3)    │                    │    부대1 (6x3)  │
│  ○○○○○○      │                    │      ○○○○○○  │
│  ○●○○●○      │                    │      ○●○○●○  │
│  ○○○○○○      │                    │      ○○○○○○  │
└─────────────────┘                    └─────────────────┘
┌─────────────────┐                    ┌─────────────────┐
│  부대2 (6x3)    │                    │    부대2 (6x3)  │
│  ○○○○○○      │                    │      ○○○○○○  │
│  ○●○○●○      │                    │      ○●○○●○  │
│  ○○○○○○      │                    │      ○○○○○○  │
└─────────────────┘                    └─────────────────┘
```

- 각 부대는 6x3 그리드에 18명의 모험가 배치
- 총 2부대 vs 2부대 전투 (최대 36명 vs 36명)
- 플레이어는 36명의 다양한 캐릭터를 수집 가능
- 완전 자동 전투 진행

2. `Assets/Scenes/Main.unity` 씬 로드
3. Play 버튼 클릭

#### CSV 데이터 생성
1. Unity 메뉴 → `GuildMaster` → `Generate CSV Data`
2. 원하는 데이터 타입 선택
3. `Assets/CSV/` 폴더에 자동 생성

#### 디버그 모드 (개발자용)
- **F1**: 디버그 패널 토글
- **1-5**: 자원 추가 (각 1000)
- **A**: 랜덤 모험가 추가
- **B**: 테스트 전투 시작
- **S**: 게임 속도 변경 (1x/2x/4x)
