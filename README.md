## 🎮 프로젝트 개요
- **게임 장르**: Tower Defense + 실시간 PvP 하이브리드 모바일 게임
- **게임 제목**: Twelve
- **개발 시작일**: 2025-06-18
- **Unity 버전**: 6000.2.0b5
- **대상 플랫폼**: 모바일 (Android/iOS)
- **게임 모드**: PvP(플레이어 대 플레이어) 및 PvE(플레이어 대 AI)
- **그래픽 스타일**: 2.5등신 SD 캐주얼 스타일
- **게임 분위기**: 병맛 유머 → 감동적 엔딩

### 🎯 게임 컨셉
Twelve는 명일방주의 타워 디펜스와 클래시로얄의 실시간 대전을 결합한 전략 게임입니다. 플레이어는 타일 기반 맵에서 캐릭터를 소환하고 배치하여 몬스터 웨이브를 방어하는 동시에, 상대방의 성을 공격해야 합니다.

### 🎨 게임 특징
- **쉬운 접근성**: 원버튼 소환 시스템으로 누구나 쉽게 시작
- **깊이 있는 전략**: 3종족 시너지, 라인 운영, 합성 타이밍 등 다양한 전략 요소
- **실시간 대전**: AI와 동일한 조건에서 경쟁하는 공정한 대전
- **차별화 요소**: 타워형 캐릭터가 이동하며 전투하는 독특한 시스템

### 🏰 게임 필드 이해하기

#### **3라인 시스템**
```
[왼쪽 라인]  [중앙 라인]  [오른쪽 라인]
    ↓          ↓          ↓
 중간성1    중간성2    중간성3
 (500HP)   (500HP)   (500HP)
    ↓          ↓          ↓
        → [최종성] ←
          (1000HP)
```

#### **배치 전략**
- **왼쪽**: 탱커 위주 (근접 전사, 방어력 높은 캐릭터)
- **중앙**: 딜러 위주 (원거리 공격수, 마법사)  
- **오른쪽**: 서포터 위주 (힐러, 버프 캐릭터)

### ⚔️ 캐릭터 시스템 완전 이해

#### **종족별 특성**

**🛡️ Human (인간) - 균형과 적응**
- **특징**: 균형잡힌 스탯, 다양한 역할 수행 가능
- **대표 캐릭터**:
  - 인간 전사: 근접 탱커 (MeleeAttack + Tank)
  - 인간 궁수: 원거리 딜러 (RangedAttack + Kiting)  
  - 인간 마법사: 마법 서포터 (MagicAttack + Support)
- **플레이 스타일**: 안정적이고 예측 가능한 전략
- **추천 조합**: 전사 2명 + 궁수 2명 + 마법사 1명

**⚔️ Orc (오크) - 압도적 화력**
- **특징**: 높은 공격력과 체력, 낮은 공격속도
- **대표 캐릭터**:
  - 오크 전사: 강력한 탱커 (MeleeAttack + Berserker)
  - 오크 투척병: 중거리 딜러 (RangedAttack + Aggressive)
  - 오크 족장: 리더 버퍼 (MeleeAttack + Leader + Buff)
- **플레이 스타일**: 공격적이고 직접적인 전략
- **추천 조합**: 족장 1명 + 전사 3명 + 투척병 1명

**🌿 Elf (엘프) - 우아한 정밀함**
- **특징**: 긴 사거리와 빠른 공격속도, 낮은 체력
- **대표 캐릭터**:
  - 엘프 레인저: 장거리 저격수 (RangedAttack + Mobility)
  - 엘프 힐러: 치유 전문가 (Support + Healer)
  - 엘프 현자: 마법 제어사 (MagicAttack + Teleport + Wisdom)
- **플레이 스타일**: 정밀하고 전술적인 전략
- **추천 조합**: 현자 1명 + 레인저 2명 + 힐러 2명

### 🎲 핵심 게임플레이 메커니즘

#### **1. 소환 시스템**
```
소환 버튼 클릭 → 미네랄 30 소모 → 랜덤 캐릭터 획득

확률:
- 1성 캐릭터: 70%
- 2성 캐릭터: 25%  
- 3성 캐릭터: 5%

종족 확률:
- Human: 40%
- Orc: 35%
- Elf: 25%
```

#### **2. 합성 시스템**
```
같은 캐릭터 3개 → 상위 등급 1개

합성 조합:
- 1성 × 3 = 2성 × 1
- 2성 × 3 = 3성 × 1

자동 합성 (주사위 버튼):
- 전체 필드를 스캔하여 합성 가능한 조합 자동 실행
- 1성 → 2성 우선 처리
```

#### **3. 배치 시스템**
```
캐릭터 드래그 → 타일에 드롭 → 자동 배치

배치 규칙:
- 한 타일당 최대 3개 캐릭터 스택 가능
- 같은 캐릭터만 스택 가능
- 스택 시 공격력 보너스 획득
```

#### **4. 종족 시너지**
```
같은 종족 캐릭터 개수에 따른 버프:

3명: 공격력 +10%, 체력 +10%
5명: 공격력 +20%, 체력 +15%  
7명: 공격력 +30%, 체력 +20%
9명: 공격력 +50%, 체력 +30%
```

### 🎯 단계별 플레이 가이드

#### **초반 전략 (1-5웨이브)**
```
1) 첫 소환으로 탱커 확보
   - 인간 전사나 오크 전사 우선

2) 두 번째 소환으로 딜러 확보  
   - 인간 궁수나 엘프 레인저 우선

3) 세 번째 소환으로 서포터 확보
   - 인간 마법사나 엘프 힐러 우선

4) 배치 위치:
   - 탱커: 가장 앞줄 (적 진로 차단)
   - 딜러: 중간줄 (사거리 활용)
   - 서포터: 뒷줄 (안전한 위치)
```

#### **중반 전략 (6-15웨이브)**
```
1) 종족 통일 시작
   - 주력 종족 선택 (Human/Orc/Elf 중 1개)
   - 해당 종족 캐릭터만 소환/배치

2) 합성 활용
   - 같은 캐릭터 3개가 모이면 즉시 합성
   - 주사위 버튼으로 자동 합성 활용

3) 5웨이브마다 보상 선택
   - 주력 종족에 맞는 2성 캐릭터 선택
   - 부족한 역할군 우선 (탱커 → 딜러 → 서포터)

4) 리사이클 활용
   - 다른 종족 캐릭터를 주력 종족으로 변환
   - 골드 50 소모하지만 시너지 효과로 상쇄
```

#### **후반 전략 (16-20웨이브)**
```
1) 최종 조합 완성
   - 3성 캐릭터 2-3개 확보
   - 같은 종족 7-9명 배치로 최대 시너지

2) 고급 전술 활용
   - 캐릭터 위치 미세 조정
   - 적 패턴에 따른 실시간 재배치

3) 보스 대비
   - 15웨이브 보스: 고화력 집중 딜러 배치
   - 20웨이브 최종 보스: 모든 라인 균등 배치

4) 미네랄 관리
   - 여유 미네랄로 추가 소환보다는 강화 우선
   - 종족별 강화 버튼 활용 (공격력/공격속도 +5%)
```

### 🏆 고급 전략 및 팁

#### **종족별 최적 조합**

**Human 덱 (안정형)**
```
핵심 캐릭터:
- 인간 대마법사 (3성): 광역 딜러 + 시간 조작
- 인간 기사 (2성): 탱커 + 돌진 공격
- 인간 궁수 (1성): 원거리 딜러

배치 전략:
- 전방: 기사 2명 (탱킹)
- 중앙: 궁수 3명 (딜링)  
- 후방: 대마법사 1명 (광역 딜링)

특징: 균형잡힌 공수, 안정적인 웨이브 클리어
```

**Orc 덱 (공격형)**
```
핵심 캐릭터:
- 오크 전쟁군주 (3성): 광역 탱커 + 부족 버프
- 오크 광전사 (2성): 극한 딜러 + 광란 상태
- 오크 전사 (1성): 기본 탱커

배치 전략:
- 전방: 전쟁군주 1명 + 전사 2명
- 중앙: 광전사 2명  
- 후방: 주술사 1명 (버프)

특징: 압도적 화력, 빠른 웨이브 클리어
```

**Elf 덱 (제어형)**
```
핵심 캐릭터:
- 엘프 세계수의 수호자 (3성): 광역 힐링 + 자연 분노
- 엘프 현자 (2성): 마법 딜러 + 순간이동
- 엘프 레인저 (1성): 장거리 저격

배치 전략:
- 전방: 검무용사 2명 (우아한 탱킹)
- 중앙: 레인저 2명 (저격)
- 후방: 현자 1명 + 수호자 1명

특징: 정밀한 컨트롤, 장기전 특화
```

#### **상황별 대응법**

**적 탱커가 많을 때**

- 관통 공격 캐릭터 위주 배치

- 광역 공격 캐릭터로 뒤의 딜러들까지 제거
```

**적 딜러가 많을 때**  
```
- 탱커 비중 증가로 어그로 분산
- 힐러 배치로 지속력 확보
- 빠른 합성으로 탱커 강화
```

**보스 웨이브 대응**
```

10웨이브 보스: 광역 패턴 대응

15웨이브 보스: 미니언 소환 대응

20웨이브 최종보스: 3페이즈 대응

# 🎮 Twelve – 통합 마스터 기획서

## 2. 핵심 게임플레이 시스템

### 2.1. 게임 플로우 및 승리/패배 조건
- **게임 플로우**: `[사전 준비: 덱 구성/업그레이드] → [인게임 전투: 웨이브 방어/실시간 대전] → [결과/보상: 캐릭터 획득/다음 전투 준비]`
- **승리 조건**:
    1.  **방어 승리**: 20웨이브를 모두 성공적으로 방어
    2.  **공격 승리**: 상대 AI의 최종성 파괴
- **패배 조건**:
    1.  자신의 최종성이 파괴됨
    2.  일부 스테이지의 제한 시간 초과

### 2.2. 소환 및 합성 시스템
- **원 버튼 소환**:
    - **비용**: 미네랄 30 소모
    - **결과**: 덱에 구성된 10종의 캐릭터 중 1명이 랜덤으로 선택되어 비어있는 타일에 랜덤 배치
    - **소환 제한**: 플레이어/AI 각각 최대 **50마리**까지 필드에 소환 가능
- **합성 (Merge) 시스템**:
    - **규칙**: 같은 등급의 동일한 캐릭터 3개가 모이면 자동으로 상위 등급 캐릭터 1개로 합성 (예: 1성×3 → 2성)
    - **위치**: 합성된 캐릭터는 재료가 된 3개의 캐릭터 중 가장 뒤쪽 캐릭터의 위치에 생성
- **자동 합성 (주사위 버튼)**:
    - 게임 화면 하단의 주사위 아이콘 클릭 시, 필드 전체를 스캔하여 합성 가능한 모든 조합을 자동으로 실행 (1성 우선)

### 2.3. 전투 및 이동 시스템
- **3라인 웨이포인트 시스템**:
    - 모든 캐릭터는 소환된 위치(왼쪽/중앙/오른쪽)에 따라 지정된 웨이포인트 경로를 따라 상대 진영으로 자동 이동
    - 캐릭터 드래그를 통해 실시간으로 라인 변경 가능
- **타겟팅 우선순위**:
    1.  **적 탐지**: 사정거리 내 적이 있으면 이동을 멈추고 공격. 없으면 웨이포인트 이동 계속
    2.  **중간성 공격**: 웨이포인트 경로 끝에 도달하면 해당 라인의 중간성(500HP)을 공격
    3.  **최종성 공격**: 중간성이 파괴되면 최종성(1000HP)을 목표로 이동
- **타워 스택 시스템**:
    - 한 타일(칸)에 동일한 캐릭터를 최대 **3개**까지 중첩(스택)하여 배치 가능
    - 스택 수에 따라 시각적 배치가 조정되며(1개→중앙, 2개→좌우, 3개→삼각형), 스택 당 공격력 보너스 획득

### 2.4. 종족, 덱, 시너지 시스템
- **3종족**: **휴먼(Human)**, **오크(Orc)**, **엘프(Elf)**
- **덱 구성**:
    - 게임 시작 전, 총 10칸의 덱을 구성해야 함
    - **구성 규칙**: 휴먼 3칸, 오크 3칸, 엘프 3칸 + 자유 1칸
- **종족 시너지**:
    - 필드에 배치된 같은 종족 캐릭터 수에 따라 강력한 버프 자동 적용
    - **3명**: 공격력 +10%, 체력 +10%
    - **5명**: 공격력 +20%, 체력 +15%
    - **7명**: 공격력 +30%, 체력 +20%
    - **9명**: 공격력 +50%, 체력 +30%
- **리사이클 시스템 (종족 변환)**:
    - 로비에서 50골드를 소모하여 특정 종족 캐릭터를 다른 2종족 중 하나로 랜덤 변환 가능 (시너지 구성에 핵심)

### 2.5. 성장 및 강화 시스템
- **1. 인게임 전 업그레이드 (레벨업)**:
    - 로비에서 **같은 종족/등급의 캐릭터**와 **10골드**를 재료로 사용하여 대상 캐릭터의 경험치를 1% 올림
    - 경험치 100% 달성 시 1레벨업하며, 모든 스탯 5% 증가 (최대 30레벨)
- **2. 인게임 강화 (종족 버프)**:
    - 전투 중, 화면 하단의 종족별 강화 버튼을 클릭
    - 미네랄을 소모하여 해당 종족의 모든 아군 캐릭터에게 즉시 공격력/공격속도 5% 버프 부여 (최대 10레벨)

## 3. 게임 콘텐츠

### 3.1. 캐릭터 및 종족 특성
- **🛡️ Human (인간) - 균형과 적응**: 균형 잡힌 스탯으로 어떤 상황에도 대처 가능. 안정적인 플레이에 적합.
- **⚔️ Orc (오크) - 압도적 화력**: 높은 공격력과 체력을 바탕으로 한 공격적인 플레이 스타일.
- **🌿 Elf (엘프) - 우아한 정밀함**: 긴 사거리와 빠른 공격속도를 활용한 정밀한 컨트롤과 장기전에 특화.

### 3.2. 스테이지 및 웨이브
- **스테이지**: 점진적으로 난이도가 상승하는 총 **100개**의 PvE 스테이지
- **웨이브**: 각 스테이지는 총 **20웨이브**로 구성
    - **일반 웨이브**: 몬스터 5마리씩 3개 라인에 분산되어 등장
    - **보스 웨이브**: 5, 10, 15, 20 웨이브에 강력한 보스 몬스터 등장
- **5웨이브 보상**: 5, 10, 15 웨이브 클리어 시, 랜덤 2성 캐릭터 3개 중 1개를 30초 내에 선택하여 즉시 획득

### 3.3. 재화 및 획득
- **미네랄**: 인게임 재화. 캐릭터 소환 및 강화에 사용
- **골드**: 로비 재화. 캐릭터 뽑기, 업그레이드, 리사이클에 사용
- **다이아몬드**: 유료 재화.
- **캐릭터 획득 (가챠)**:
    - **단일 뽑기**: 100골드
    - **10연차 뽑기**: 900골드 (10% 할인)
    - **확률**: 1성(70%), 2성(25%), 3성(5%)

## 5. 플레이어 게임플레이 완전 가이드

### 5.1. 게임 시작 전 준비 (로비)
1.  **스테이지 선택**: 100개의 스테이지 중 클리어한 다음 스테이지를 선택합니다.
2.  **덱 구성**: **가장 중요합니다.** 인벤토리에서 캐릭터를 드래그하여 총 10칸의 덱을 채웁니다. (휴먼 3, 오크 3, 엘프 3, 자유 1 규칙 준수)
3.  **캐릭터 강화**: 업그레이드, 리사이클 기능을 활용하여 덱을 최적화합니다.
4.  **캐릭터 획득**: 골드가 부족하면 가챠를 통해 새로운 캐릭터를 뽑습니다.

### 5.2. 단계별 플레이 전략

#### **초반 (1~5 웨이브): 필드 구축**
- **목표**: 빠르게 필드를 채워 안정적인 방어선 구축
- **행동**: 미네랄이 모이는 대로 **소환** 버튼을 눌러 캐릭터 수를 확보합니다. 최소 10~15마리를 목표로 합니다.
- **팁**: 첫 탱커(인간/오크 전사)와 첫 원거리 딜러(인간 궁수/엘프 레인저) 확보가 중요합니다.

#### **중반 (6~15 웨이브): 질적 성장**
- **목표**: 주력 종족을 정하고 시너지를 극대화하며, 고급 캐릭터 확보
- **행동**: 소환과 **합성**을 병행합니다. 같은 캐릭터 3개가 모이면 즉시 합성하여 2성, 3성 유닛을 만듭니다. **자동 합성(주사위)** 버튼을 적극 활용하세요.
- **팁**: 5, 10, 15 웨이브 보상 선택 시, 현재 자신의 덱에 가장 필요한 종족/역할의 2성 캐릭터를 선택하여 성장의 발판으로 삼으세요.

#### **후반 (16~20 웨이브): 최종 강화 및 마무리**
- **목표**: 완성된 조합으로 최종 보스 격파
- **행동**: 소환보다는 **인게임 강화**에 미네랄을 투자합니다. 주력 종족의 강화 버튼을 눌러 스펙을 극한으로 끌어올립니다.
- **팁**: 필드가 꽉 찼다면 더 이상 소환하지 말고, 라인 이동과 스택 시스템을 활용해 배치를 최적화하는 데 집중하세요.

### 5.3. 고급 전략 및 팁
- **타워 스택의 이해**: 합성이 애매할 때, 같은 캐릭터를 한 타일에 3개까지 쌓아(스택) 특정 라인의 화력을 극대화할 수 있습니다. 이는 강력한 단일 적이나 보스를 상대할 때 유용합니다.
- **리사이클은 투자**: 당장 골드가 아까워도, 덱에 맞지 않는 종족을 리사이클하여 주력 종족으로 바꾸는 것은 후반 시너지 효과를 생각하면 최고의 투자입니다.
- **AI 패턴 예측**: AI는 초반에 균형, 중반에 효율(합성), 후반에 공격적인 패턴을 보입니다. AI보다 한발 앞서 종족 시너지를 완성하거나, AI의 주력 라인을 예측하여 카운터 유닛을 배치하세요.
