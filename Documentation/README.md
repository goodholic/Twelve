🏰 GuildMaster - 정예 부대 키우기

   

장르: 스토리 기반 + 캐릭터 수집형 턴제 자동 방치 RPG

플랫폼: 모바일 (iOS/Android), PC(스팀)

화면: 세로 모드 전용 (1080x1920 기준)

플레이: 완전한 싱글플레이어 게임

🎮 게임 소개
매력적인 스토리를 따라 18명의 정예 모험가 부대를 이끌고 성장하며, 앞을 가로막는 강적들을 격파해나가는 방치형 부대 전투 RPG입니다.

핵심 특징
2:2 부대 전투: 각 부대당 9명씩, 총 18명의 캐릭터로 정예 전투

캐릭터 수집: 18명의 다양한 캐릭터를 수집하고 육성

완전 자동화: 설정 후 자동으로 진행되는 방치형 시스템

스토리 중심 진행: 모든 전투와 성장이 스토리를 따라 진행됨

싱글플레이: 타인과의 경쟁 없이 자신만의 속도로 즐기는 게임
